hexo搭建个人博客（一）

具体的步骤上面的视频和文章已经写得很详细了，但因为我这里是windows系统，遇到了和上面不一样的问题，包括

第一个坑

到部署至GitHub这一步时，运行至hexo d时出现如下报错（这里我用的还是cmd）

网上查了很久，大概是git没有配置环境变量的问题，可选择重新添加配置，或者直接下载Git，在git bash运行。基于各种因素（bash命令行对我更友好以及开头因为新上手刚好想调整blog文件位置等原因），选择直接下载Git

可自行前往Git官网下载

但国内一般贼慢，比如我这里直接进不去，所以找到了这个 Git镜像

安装步骤和普通软件一样，可自行参考 Windows10下安装Git

第二个坑

在.\blog\_config文件中配置完自己GitHub仓库信息后直接运行hexo d出现

其实图中已经给提示了，需要先tell it who I am，在hexo d之前敲两行

$ git config --global user.email "你的GitHub注册邮箱"

$ git config --global user.name "你的GitHub昵称"
